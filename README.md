# 4️⃣팀원소개

- 강민수: 팀장, 화이트보드, 코스 추천 알고리즘
- 이재욱: 메뉴 UI, 실시간 화상채팅
- 김혜인: 로그인 및 인증, UI 디자인, DB
- 민경욱: DB, 목적지 검색, 실시간 화상채팅, 코스 추천, OAuth/JWT
- 김수빈: DB 및 API 구축, 인프라 설계 및 구축, 운영 환경 배포
